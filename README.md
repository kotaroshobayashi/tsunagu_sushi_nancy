# Nancy Project README

## 概要

このフォルダは、`TSUNAGU` としてフランスでの飲食出店機会を調査・準備するためのプロジェクト管理用ディレクトリです。
現在の主案件は、`Appel à candidature FOOD TRUCK - NOA 2026 (1)` に基づく `Nancy Open Air 2026` へのフード出店応募です。

本プロジェクトでは、日本風ストリートフードを軸に、フランス国内フェス・イベントへの出店を目指しています。会話ログを見る限り、現時点では複数イベントを比較検討したうえで、まず `Nancy Open Air 2026` を中心案件として進めています。

## 背景

- フランスでの期間限定出店・フェス出店を実現するための実証案件。
- 出店形態は、現時点ではフードトラックというより `テント形式` を前提に応募準備が進んでいる。
- 提供予定メニューは、日本のストリートフード文脈を意識した構成。
- 主催者からは、応募書類に加えて営業・衛生・保険・社会保障関連の証明書類提出が求められている。
- プロジェクトの運営は少人数チームで進んでおり、役割分担、収支試算、必要許認可の確認が並行で進んでいる。

## 会話ログから見えるプロジェクトの流れ

参照ログ:

- `[LINE]🇫🇷.txt`

主な流れ:

- 2026年3月末時点で、チーム立ち上げと役割の話し合いが始まった。
- Nancy 以外にも複数フェスを調査したが、締切や条件の問題で候補がかなり絞られた。
- `HACCP` は取得可能性が高く、費用感も概ね把握できている。
- 4月2日前後の会話で、練習場所、初期費用の分担、利益分配、渡航日程などの大枠方針が一度整理された。
- その後、`Nancy Open Air` 主催側から追加提出書類の依頼が来ており、案件は調査段階から実務段階に移行している。
- 一方で、収支試算、許認可理解、出資と利益分配の整理、メンバー間の情報共有にはまだ揺れがある。

## AIアシスタントの想定役割

このプロジェクトでは、AIは単なるチャット相手ではなく、`全体状況の可視化` と `書類・進行管理の支援` を行うアシスタントとして機能させる想定です。

想定業務:

- 会話ログの要約
- 現在の決定事項と未決事項の整理
- 提出書類チェックリストの作成
- 対外メール案の作成
- ミーティング用アジェンダの作成
- メンバーごとのタスク整理
- `LINE` 上での進捗報告・リマインド対応
